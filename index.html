metaConvert

Automated computations of effect sizes

metaConvert is a no-code, free and open-source tool facilitating effect size computations. More information on the calculations used in this tool can be found in its dedicated CRAN or github page

Open the app
Unique features
metaConvert esc compute.es esci metafor RevMan CMA
Automatic workflow
Number of effect measures
Conversion between effect measures
Aggregation of ES in multilevel situations
Multiple ES estimations per comparison
Dataset comparison
Automatically flip effect size direction
Meta-analytic calculations
Graphical-User Interface (no coding required)
