WARNING

This section describes all the combination of input data allowing to estimate and/or convert the 11 effect size measures available in metaConvert.

Importantly, it is not because you can obtain the same effect size measure from various input data that it makes sense to pool them in a meta-analysis.

Exemple: The effect size measure of your meta-analysis is SMD. Even if metaConvert gives you the possibility, it would (generally) have very limited meaning to pool a SMD obtained from a RCT change scores with a SMD obtained by converting a correlation coefficient from a cross-sectional study.

Use the tables in this section when drafting your protocol: pre-specify the input data you will use to estimate your target effect size measure of your meta-analysis.
